# Projeto Acadêmico - DSSW

Este repositório contém o projeto desenvolvido para a disciplina de **Desenvolvimento de Sistemas Web**, do 5º semestre do curso de **Tecnologia em Sistemas para Internet (TSI)** no **IFSP** (Instituto Federal de Educação, Ciência e Tecnologia de São Paulo).

## Estrutura do Projeto

O projeto é dividido em duas partes principais:

- **`/backend`**: API robusta desenvolvida em Node.js utilizando o framework NestJS.
- **`/frontend`**: Aplicação de interface com o usuário construída utilizando Vite.

## Como Executar

### Backend
1. Navegue até a pasta `backend`: `cd backend`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run start:dev`

### Frontend
1. Navegue até a pasta `frontend`: `cd frontend`
2. Instale as dependências: `npm install`
3. Inicie a aplicação: `npm run dev`
